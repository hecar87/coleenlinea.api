# Service Prototype

> Version: 1.0.0

Este es el proyecto base para la implementación de los microservicios del Sistema. Este proyecto cuenta con la configuración básica para comenzar rápidamente el desarrollo de la lógica.

## Dependencias Instaladas

### Intervention Image
Libreria para la manipulación de imagenes. [Documentacion](https://image.intervention.io/v3/introduction/installation)

### AMQP Lib
Librería que permite la comunicación mediante el protocolo AMQP. [Documentación](https://github.com/php-amqplib/php-amqplib)

### Bschmit Lib
Libreria que facilita la publicación y consumo de mensages con protocolo AMQP desde Rabbit MQ. [Documentación](https://github.com/bschmitt/laravel-amqp)
